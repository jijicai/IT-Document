# 1、Eclipse 介绍

    Eclipse 是一个开放源代码的、基于Java的可扩展开发平台。
    它只是一个框架和一组服务。
    它通过插件组件构建开发环境。
    它发布版本时会默认附带插件集。
    它可以作为多种语言的集成开发环境（IDE），例如：Java、Kotlin、Scala、C/C++、COBOL、PHP、Android（Java）等。

它可以在 Mac OS X、Windows、Linux 操作系统上使用，最新版本是：Eclipse IDE 2019-09。最新版本默认情况下，使用安装器安装 Eclipse。
